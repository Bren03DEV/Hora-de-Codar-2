# 📘 Hora de Codar 2

Este repositório contém exercícios de **lógica de programação desenvolvidos em JavaScript**, com foco na prática de conceitos fundamentais da linguagem e no desenvolvimento do raciocínio lógico!

Os programas utilizam **entrada de dados pelo usuário** (`prompt`) e **exibição de resultados** (`alert` ou `console`), abordando situações comuns como comparações, cálculos matemáticos e tomadas de decisão.

## 🚀 Tecnologias

* JavaScript
* HTML

## 🧠 Conteúdo Praticado

* Estruturas condicionais (`if`, `else if`, `else`)
* Operadores lógicos e matemáticos
* Manipulação de números (`parseInt`, `parseFloat`)
* Cálculo de médias
* Comparação de valores
* Pequenas aplicações como **calculadora simples** e **cálculo de peso ideal**

## 🎯 Objetivo

Praticar **fundamentos de programação e lógica utilizando JavaScript**, resolvendo uma lista de exercícios que envolvem operações matemáticas, validações e decisões baseadas em dados informados pelo usuário.
